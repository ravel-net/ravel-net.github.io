## Usage

1. install Jekyll
2. Clone/pull, Jekyll, and push

	$ git clone https://github.com/ravel-net/ravel-net.github.io.git
	$ cd ravel-net.github.io.git
	$ jekyll s
	# preview website locally at localhost:4000

	$ git pull
	# local edits
	$ git add .
	$ git push
